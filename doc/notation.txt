- Assuming all pastures are identified 0-31
- Hex directions are labeled A-F, clockwise

Notation Options:
1. Relative connections
PID 1 - Dir - PID 2, ...
Example: 0A30,1B10

2. Adjacency Matrix table
PID 0: A, B, C, D, E, F|
PID 1: A, B, C, D, E, F|
...
Example: 1,30,3,10,5,6|2,3,4,5,7|

3. Coordinates: (x32)
X,Y|...
Example: 3,1|,10,5
Note: This might require normalization
